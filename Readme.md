# Notes PWA — Приложение для заметок

Прогрессивное веб-приложение (PWA) для управления заметками с поддержкой офлайн-режима, push-уведомлений и напоминаний в реальном времени. Разработано в рамках практических занятий 13-18 по дисциплине "Фронтенд и бэкенд разработка".

## Технологии

- Frontend: HTML, CSS, JavaScript, Chota CSS
- Backend: Node.js, Express, Socket.IO
- PWA: Service Worker, Web Push API, PushManager
- Безопасность: HTTPS (mkcert), VAPID-ключи
- Хранение данных: localStorage

## Практические занятия

### Практическое занятие 13-14 — Базовое PWA

Реализовано базовое прогрессивное веб-приложение для управления заметками.

Выполнено:
- Создание HTML-страницы с формой добавления заметок и списком заметок
- Хранение заметок в localStorage браузера
- Добавление и удаление заметок без перезагрузки страницы
- Подключение манифеста PWA (manifest.json) с иконками и метаданными приложения
- Регистрация Service Worker для работы приложения в офлайн-режиме
- Кэширование статических ресурсов при первом посещении
- Отображение индикатора офлайн-режима при отсутствии интернета
- Стратегия Cache First для загрузки ресурсов из кэша

### Практическое занятие 15 — HTTPS + App Shell

Обеспечена безопасность приложения с помощью локального HTTPS и реализована архитектура App Shell.

Выполнено:
- Установка утилиты mkcert и генерация самоподписанного SSL-сертификата для localhost
- Запуск приложения по протоколу HTTPS на локальной машине
- Переработка структуры приложения под архитектуру App Shell
- Вынос динамического контента в отдельные HTML-фрагменты (content/home.html, content/about.html)
- Реализация навигации между страницами через динамическую загрузку контента с помощью fetch
- Добавление страницы "О приложении"
- Обновление Service Worker: стратегия Cache First для статики и Network First для динамического контента
- Кэширование динамических страниц в отдельный кэш (dynamic-content-v1)
- Поддержка офлайн-режима для динамических страниц через фолбек на home.html

### Практическое занятие 16 — WebSocket + Push-уведомления

Реализована двусторонняя связь в реальном времени через WebSocket и система push-уведомлений.

Выполнено:
- Создание Node.js сервера на базе Express с поддержкой HTTPS
- Подключение библиотеки Socket.IO для работы с WebSocket
- Рассылка события всем подключённым клиентам при добавлении новой заметки
- Отображение всплывающего сообщения при получении заметки от другого клиента
- Генерация VAPID-ключей для идентификации сервера при отправке push-уведомлений
- Реализация подписки и отписки на push-уведомления через PushManager
- Добавление кнопок "Включить уведомления" и "Отключить уведомления"
- Сохранение push-подписок на сервере и отправка уведомлений через web-push
- Добавление обработчика push-событий в Service Worker
- Управление состоянием уведомлений через postMessage между клиентом и Service Worker

### Практическое занятие 17 — Детализация Push, напоминания

Расширен функционал push-уведомлений: добавлены напоминания с возможностью откладывания.

Выполнено:
- Добавление формы для создания заметки с указанием даты и времени напоминания
- Изменение структуры данных заметок в localStorage: добавлен уникальный идентификатор и поле reminder
- Отображение времени напоминания в списке заметок
- Планирование push-уведомлений на сервере с помощью setTimeout
- Хранение активных таймеров напоминаний в Map на сервере
- Добавление кнопки "Отложить на 5 минут" (Snooze) в push-уведомление
- Реализация эндпоинта /snooze для переноса напоминания на 5 минут
- Обработка нажатия на кнопку уведомления в Service Worker через событие notificationclick
- Отправка запроса на сервер из Service Worker при откладывании напоминания
- Работа напоминаний при закрытой вкладке браузера

## Установка и запуск

### 1. Установи зависимости

```bash
npm install express socket.io web-push body-parser cors
```

### 2. Создай SSL сертификат

```bash
mkcert -install
mkcert localhost 127.0.0.1 ::1
```

### 3. Запусти сервер

```bash
node server.js
```

### 4. Открой в браузере
https://localhost:3001

## Использование

1. Открой приложение по адресу https://localhost:3001
2. Нажми "Включить уведомления" и разреши уведомления в браузере
3. Добавляй заметки через форму "Добавить"
4. Для напоминания заполни форму "Добавить с напоминанием", укажи текст и дату/время
5. В указанное время придёт системное уведомление с кнопкой "Отложить на 5 минут"
6. Открой две вкладки — при добавлении заметки во второй вкладке появится всплывающее сообщение
